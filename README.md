# 企查查多账号管理扩展 (QCC Account Manager)

一款功能强大的 Chrome 浏览器扩展，专为管理企查查 (qcc.com) 本地多账号会话设计。支持账号无缝切换、自动化保活续期以及基于 WebDAV 的多设备跨端增量同步。

## ✨ 核心功能

### 🔗 一键多账号切换

- **保存会话**：一键保存当前已登录的企查查账号状态（包括 Cookies 和 LocalStorage）。
- **无缝切换**：在已保存的多个账号间一键切换。切换时会自动清空旧有状态并注入目标账号的数据，页面自动刷新即可生效，免去频繁扫码登录的烦恼。
- **痕迹清理**：提供“清空痕迹并录入新账号”功能，安全清理当前浏览器环境，以供全新的账号登录录入。

### 🛡️ 状态监测与测活

- **一键测活**：批量查验列表内所有账号是否在线（精确识别 401/403 失效与 425 封控限制）。
- **实时监控**：在弹窗界面直观显示当前页面是否遭遇企查查极验验证码、访问受限或是调用频额超限。

### 🔄 自动化保活续期

- **当前账号保活**：插件会在后台静默定期访问企查查首页，保持当前使用中账号的 Session 活跃，避免因长期防呆或不操作被踢出。
- **过期告警系统**：随时监控所有已存备用账号的有效期限。如发现非当前登录账号的剩余 Cookie 有效期不足 30 天，将触发桌面级通知告警，提醒您及时切换保活。

### ☁️ WebDAV 多端云同步

- **目录模式双向同步**：通过轻量级的 `manifest.json` 和独立的 `{uuid}.json`，实现极速的轻量化增量同步。
- **多设备漫游**：无论在办公电脑还是个人笔电，只需连入您私有的 NAS (如 Synology) 或 VPS 上的 WebDAV 服务，即可随时拉取最新账号池。后台会自动双向同步有变更的数据（如最新的保活时间）。
- **手动控制极简**：提供一键 ⬇️ 从云端同步（仅合并更新资源）与 ⬆️ 推送云端（增量提交）。

## 🚀 快速使用说明

1. 下载或 Clone 本项目，在 Chrome 扩展管理 `chrome://extensions/` 开启 **开发者模式**。
2. 点击 **加载已解压的扩展程序**，选取本项目文件夹。
3. （重要）右下角点击 **⚙️配置** 图标，填入您的私有 WebDAV 服务器地址及账号密码。
4. 打开企查查 `qcc.com` 并正常登录一个账号，点击弹窗中的“保存当前账号信息”。
5. 如果需要录入新账号，点击“录入新账号”，页面刷新还原未登录状态后，登录新账号再次保存即可体验一键无缝切换和跨端 WebDAV 同步。

## 🔒 隐私与安全性申明

本项目严格遵循本地离线及安全开发标准：

1. **0 收集**：所有账号信息、Cookie 等敏感凭证均仅保存在您本地，以及您指定的**私有 WebDAV** 服务器中。
2. **0 拦截**：代码透明，不内置任何外围服务器探针接口。所有的保活及网络验证请求，均基于您的本地当前公网 IP 直接向企查查及自身 WebDAV 站点发起。
